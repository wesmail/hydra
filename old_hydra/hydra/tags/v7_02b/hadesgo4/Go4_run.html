##################### RUNNIG THE MONITOR ####################

    If you have managed to setup your configuration , you can run
    the Monitor. Go4 will be the server part and the Analysis will
    be the client. You can run both on the same machine or on different
    ones.

    Go4 has to be started first and then the Client.
    The Client needs the following arguments to start:

    Client MySessionName myHost myPort myParameterFile

    MySessionName= myMon (any name)
    myHost              = lxg0424 (any machine)
    myPort               = 5000  (default port of Go4,
                                              be careful, if there is already a Go4 running,
                                              Go4 will increase this number (5001 ...).
                                              This port number has to match between client and Go4 server.)

    myParameterFile= Hydradir/hadesgo4/Go4HadesClient/analysisParams.txt
                                   (or any other place or file name following the syntaxe above)


     If the ssh remote shell asks for a password you might have to generate a dsa key first.
   
    1.  . ./hadesgo4login.sh
    2.  go4&

    3.a. -------------------------------------------------------------
         To run the Client on the same machine:
         ${HADDIR_HADESGO4}/Go4HadesClient/MainGo4HadesClient MySessionName myHost 5000 'Hydradir/hadesgo4/Go4HadesClient/analysisParams.txt'

    3.b  -------------------------------------------------------------
         To run the Client remote on some other machine:
         Use the Launch analysis client dialog:

     Name        : name of your session                      (myMon)
     Host          : machine where the client will run (lxg0424)
     Dir            : path of the client executable          (Hydradir/hadesgo4/Go4HadesClient/)
     Executable: path to the config parameter file   (Hydradir/hadesgo4/Go4HadesClient/analysisParams.txt)
     switch to ssh option!

     -------------------------------------------------------------


     4.  Start User Panel on the Go4 Main GUI (Go4 Hades Monitor button will pop up)
     5.  Press the button to Open Hades Monitor main panel
     6.  Click on start if the Client is ready to run
     7.  Open any Monitor you want to look at
     8.  Open the Debugger to find out, why the hell you got "break segmentation"
     9.  Please, don't call me and debug your self.
